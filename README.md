Free Notes Summarizer

Built by Navraj Singh Waraich

A free web based application that summarizes pasted notes into concise bullet points using natural language processing techniques.

Features
- Paste notes and generate instant summaries
- Adjustable summary length
- 100% free (no API or billing required)

 Tech Stack
- Python
- Streamlit
- NLP (sentence scoring, stop-word filtering)

How to Run Locally
```bash
pip install streamlit
streamlit run app.py
